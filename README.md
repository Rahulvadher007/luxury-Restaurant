# 🍽️ Luxury Restaurant Website

A modern and elegant restaurant website built with a premium user experience in mind. The project features a luxurious design, responsive layout, interactive menu sections, reservation functionality, location integration, and an administrative management portal.

## ✨ Features

### Customer Side

* Modern luxury-themed homepage
* Responsive design for mobile, tablet, and desktop
* Hero section with restaurant branding
* Interactive food and drink menu
* About Us section
* Contact information
* Google Maps integration for restaurant location
* Smooth animations and transitions
* Reservation/Booking interface

### Admin Panel

* Secure admin login page
* Management dashboard
* Restaurant content management
* Easy navigation between website and admin portal

### UI/UX

* Elegant and professional design
* Premium typography
* Smooth scrolling experience
* Fast loading performance
* Mobile-first responsive layout

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
* Responsive Web Design
* Google Maps Integration
